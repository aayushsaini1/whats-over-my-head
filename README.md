# What's Over My Head? 

A live flight telemetry scanner and tracker built with Next.js and custom CSS. It localizes nearby air traffic using your device coordinates, manual search queries, or preset high-traffic airport hubs.

## Features

- **Live ADS-B Telemetry**: Locates active transponders in your vicinity.
- **Visual Aircraft Profiles**: Automatically retrieves aircraft photos and airline carrier logos.
- **Control Desk**: Adjustable scanning range slider, auto-scan interval toggles, and manual search/refresh.
- **Privacy First**: Geolocation coordinates are processed locally on the client only.
